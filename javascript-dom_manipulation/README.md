# JavaScript DOM Manipulation Project

## Introduction

This project covers the fundamentals of JavaScript DOM (Document Object Model) manipulation. You'll learn how to select and modify elements on a web page, make requests to fetch data, handle user interactions, and create dynamic web applications using modern client-side web APIs.

## Learning Objectives

By the end of this project, you should be able to:

- Select HTML elements using JavaScript methods.
- Differentiate between ID, class, and tag name selectors.
- Modify the style of HTML elements dynamically.
- Retrieve and update content within HTML elements.
- Manipulate the DOM by creating, appending, or removing elements.
- Make asynchronous requests using `XmlHttpRequest` and the Fetch API.
- Listen and bind to DOM events for user interactions.

## Requirements

### General

- Allowed editors: All of them.
- All your files will be interpreted on Chrome browser (version 57.0 or later).
- All your files should end with a new line.
- A mandatory `README.md` file with meaningful information about the content should be placed at the root folder of the project.
- Your code should be semistandard compliant.
- You are not allowed to use `var`.
- HTML should not reload for each action: DOM manipulation, update values, fetch data, etc.
